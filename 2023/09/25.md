## わかったこと
- gitのHEADやブランチの移動
  - git br -f feat HEAD~2
  - ~は指定したコミットから複数回移動する
  - git rebase -i HEAD~4 はHEADを含めて4つ分
  - git rebase -i HEAD^^2^のような記載も可能
  - git rebaseとamendで過去のコミットを微修正することができる
